Guía de Uso del Sistema

Perfiles: Crea múltiples perfiles para diferentes negocios o proyectos. Cada perfil tiene sus propios datos separados.

Registro: Usa los botones de colores para seleccionar tipo (Ingreso/Egreso/Promesa). Para egresos, indica si sale del Capital Total o Caja Chica.

Compartir con URL: Genera un enlace con todos los datos del perfil activo comprimidos. Otros pueden importarlos automáticamente haciendo clic en el enlace.

Backup en la Nube: Conecta tu Dropbox para guardar copias de seguridad de TODOS tus perfiles automáticas o manuales.

Exportar: Si tienes un solo perfil, exporta directamente. Con múltiples perfiles, elige exportar solo el activo o todos juntos.
